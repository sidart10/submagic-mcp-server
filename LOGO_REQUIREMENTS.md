## Design Guidelines

### âœ… DO

- **Keep it simple** - Bold, recognizable designs work best
- **Test at small sizes** - Logo will be displayed at 64Ã—64 and smaller
- **Use vector graphics when possible** - For clean, sharp edges
- **Consider the context** - Think "video editing" theme
- **Brand consistency** - Use colors that represent Submagic or video content

### âŒ DON'T

- **Avoid small text** - Won't be readable when scaled down
- **Avoid complex details** - Gets lost at thumbnail sizes
- **Avoid low contrast** - Makes logo hard to see
- **Avoid copyrighted images** - Unless you have permission
- **Avoid offensive content** - Keep it professional

## Where to Upload Logo

### GitHub

```bash
# Create assets directory
mkdir -p assets

# Add your logo
mv logo.png assets/logo.png

# Commit
git add assets/logo.png
git commit -m "Add marketplace logo"
git push
```

Your logo URL will be:
```
https://raw.githubusercontent.com/YOUR-USERNAME/submagic-mcp-server/main/assets/logo.png
```

### Alternative: Use GitHub Issues

Upload directly in your marketplace submission issue - GitHub will host it.
